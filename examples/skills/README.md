# Example Skills

These SKILL.md files were generated by [Skilldo](https://github.com/SkillDoAI/skilldo) and are provided as demonstrations of the pipeline's output. They were generated using a variety of models (Claude Sonnet 4.6, GPT-5.3 Codex, Cerebras gpt-oss-120b) with default configuration and no library-specific prompt tuning.

**These skills are for demonstration purposes only.** While we largely believe them to be accurate, there can always be improvements with per-library configuration tweaking. If you are a library owner or maintainer, you may need to adjust the prompts or configuration to get the most accurate output for your specific library.

See `examples/configs/` for sample configuration files and the [README](../../README.md) for documentation on custom instructions and per-stage prompt tuning.
